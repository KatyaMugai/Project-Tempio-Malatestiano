Enriching a Knowledge Graph: Tempio Malatestiano
Project 1: Enriching a Knowledge Graph
Abstract
This project explores the ArCo knowledge graph in order to identify possible information gaps in the RDF description of Tempio Malatestiano, an architectural heritage resource located in Rimini.
The project focuses on two main gaps:
	●	the representation of internal architectural components;
	●	the representation of historical and artistic entities associated with the monument.
Through SPARQL exploration, we identified information that is present indirectly in photographic resource labels but is not explicitly modeled as RDF relations of the main architectural resource. We then proposed RDF triples and a small vocabulary extension to enrich the knowledge graph.
